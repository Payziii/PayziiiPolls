# PayziiiPolls - Frontend

Красивый фронтенд для управления опросами с поддержкой диаграмм и статистики.

## 🎨 Особенности

- 🟢 **Зеленый + Темный дизайн** - современный интерфейс
- 📊 **Интерактивные графики** - Chart.js для визуализации результатов
- ✨ **Удобный интерфейс** - создание и управление опросами в один клик
- 📱 **Responsive** - работает на любых устройствах
- 🔗 **Публичные ссылки** - делитесь опросами через ссылки
- 📈 **Подробная статистика** - включая NPS для шкальных вопросов

## 🚀 Установка и запуск

```bash
npm install
npm run dev
```

## 📋 Структура

- `/src/views/` - страницы приложения
- `/src/components/charts/` - графики (BarChart, ScaleChart)
- `/src/api/client.js` - API клиент
- `/src/assets/styles.css` - темизация

## 🎯 Функции

1. **Создание опросов** с различными типами вопросов
2. **Типы вопросов**: один ответ, несколько, текст, шкала
3. **Поделиться опросом** через публичную ссылку
4. **Просмотр статистики** с интерактивными графиками
5. **NPS расчет** для шкальных вопросов
